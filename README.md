# Expense Manager (MERN Stack)

## Overview
This project implements a full-stack Expense Manager system using MERN architecture (MongoDB, Express.js, React.js, Node.js).

## Features
- JWT Authentication
- Secure Login & Signup
- Add and Split Expenses
- REST API architecture
- MongoDB schema for users and expenses

## Architecture
- MVC Pattern (Models, Routes, Middleware)
- RESTful API design
- Stateless authentication using JWT

## Technical Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## How to Run
1. cd server → npm install → node server.js
2. cd client → npm install → npm start
